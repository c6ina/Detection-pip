# Detection-pip
Проект для детекции пешеходов на видео с использованием нейросети YOLOv4-Tiny и библиотеки OpenCV.
Модель обрабатывает видео в реальном времени и выделяет людей зелёными рамками.

Возможности:
-Детекция только людей (person)
-Работа с видеофайлами и веб-камерой
-Реализация на OpenCV DNN (без Darknet)
-Лёгкая и быстрая модель (YOLOv4-Tiny)

Структура проекта:
├── Pedestrian_detection.py   # основной скрипт
├── coco.names                # список классов (COCO)
├── yolov4-tiny.cfg           # конфигурация модели
├── yolov4-tiny.weights       # веса обученной модели
└── video.mp4                 # входное видео (переименовать при необходимости)

Требования:
Python 3.8+
Установка зависимостей:
pip install opencv-python numpy imutils

Запуск:
1) Поместить своё видео в папку проекта
2) Указать имя видеофайла в коде: "cap = cv2.VideoCapture("video.mp4")"
3) Затем запустить python Pedestrian_detection.py
4) Окно с детекцией откроется автоматически (Выход — клавиша ESC)

Краткое описание работы программы:
1) YOLOv4-Tiny обрабатывает каждый кадр видео
2) Из всех объектов выбирается только класс person
3) Применяется Non-Maximum Suppression
4) Рисуются bounding box’ы вокруг пешеходов

Используемые технологии
-OpenCV DNN (глубокие / сверточные нейросети)
-YOLOv4-Tiny (модель обнаружения объектов)
-COCO dataset (названия выходных классов)
